# 📚 Bootstrap Typography Showcase

A simple and clean project demonstrating all major **Bootstrap Typography features** in one page.
This project is useful for beginners to understand how text styling works in Bootstrap.

---

## 🚀 Features

* Headings (`h1` to `h6`)
* Lead paragraph styling
* Text formatting (bold, italic, underline, highlight)
* Abbreviations and quotes
* Code and keyboard input styling
* Preformatted text
* Text colors and background colors
* Text alignment utilities
* Text transformation (uppercase, lowercase, capitalize)
* Lists (unstyled & inline)
* Description lists

---

## 🛠️ Technologies Used

* HTML5
* Bootstrap 5 (via CDN)
